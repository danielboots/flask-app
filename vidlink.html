  4. Deploy The Application To Heroku

    5. Before we create our Heroku application, we need to setup some files that Heroku needs to run the app. First, we need to tell Heroku which applications and dependencies are required to run our app:

    "pip3 freeze --local > requirements.txt".

    Next, the Procfile is what Heroku looks for to know which file runs the app, and how to run it, so we'll use the echo command: 

     "echo web: python app.py > Procfile".
